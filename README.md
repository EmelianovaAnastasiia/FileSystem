### Filesystem simulator

Filesystem simulator creates imitation model for linux-like file system.  

#### HOW TO: Run program
```bash
python main.py
```

#### List of commands
|Command    | Description                                                                       | Example                   |
| ------    | ---                                                                               | ---                       |
| ls        | Lists files in target directory. Runs in the current directory if nothing passed. | ``` ls ```                |
| cd        | Changes current directory.                                                        | ``` cd ../other_dir ```   |
| mkdir     | Creates new directory in current.                                                 | ``` mkdir some_dir ```    |
| touch     | Creates new file in the current directory.                                        | ``` touch file_name ```   |
